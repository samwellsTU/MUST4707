JavaScript Basics: Introduction to Arrays

What is an Array?

An array in JavaScript is a single variable used to store different elements. It is a list of items that are stored in square brackets []. Each item in an array is called an element, and each element is accessed by its numerical index.

Creating an Array

You can create an array by listing values separated by commas inside square brackets. For example: let fruits = ["apple", "banana", "cherry"];.

Accessing Array Elements

You can access elements of an array by referring to the index number. Remember, arrays in JavaScript are zero-indexed: the first element is at index 0.

Modifying Array Elements

Arrays are mutable, meaning you can change their elements. For instance, you can change the second item in the fruits array by doing fruits[1] = "blueberry";.

Array Methods and Properties

JavaScript arrays come with built-in methods and properties. For example, length gives you the number of elements in an array, and methods like push() and pop() allow you to add and remove elements.
